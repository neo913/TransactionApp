Angular Code Assignment

Angular Project prerequisite:

  • Angular 8.0+
  • Bootstrap/Material Design
  • Sass/SCSS
  • Accessibility (level AA)

Create a new project skeleton with Angular CLI 8.0+ and add following artifacts.
  • Parent component called Transaction list
  • Child component called Transaction Details
  • Transaction Service

Transaction List (parent component)

  • This component should contain following accessible table (Table 1.1).
  • User should be able to navigate to the 'Transaction Detail' (child component) by clicking on the View link under the Action column.
  • Populate the table data by invoking Transaction Service function that returns mock data.
Table 1.1 User Transactions
ID Date Comments Action
1 01/10/2020 Utility bill View
2 15/10/2020 View

Transaction Detail (child component)

  • This component should contain following information in an Angular form.
  • User should be able to update 'Comments' field and navigate back to the 'Transaction List' component.
  • New added/updated comments should be available on the Transaction List table.
Field Name Requirements
ID Read-only
Date Read-only; format should be DD/mm/yyyy
Comments Required; Editable input control; allows only alphanumeric

Transaction Service

Create a mock service that returns 'User Transactions' data as shown in Table 1.1.
